the Magnitude and Spread of the Transient Climate Response to CO<sub>2</sub> in CMIP6 Earth System Models"** ([EarthArXiv Link](https://eartharxiv.org/emgxb/))

```bash
├── README.md

│   ├── generate_figures
│   ├── make_data_dictionary
│   └── process_data
├── replicate_figures
│   ├── Figure_1.ipynb
```
